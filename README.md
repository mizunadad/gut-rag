# 腸で考えるRAG

> 腸で感じた直感を、記録し、問いを通じて言語化する

## こんな経験ありませんか？

- 会議の後、なんかもやっとしたまま終わった
- 「なんか違う気がする」けど、うまく言えなかった
- あとで考えようと思ったら、感覚ごと忘れていた

そのための記録・言語化ツールです。

## コンセプト

「腑に落ちる・気配を読む・なんか違う」——これらは脳機能の異なる部位が処理しているから感じる現象であり、言語化の前に身体が先に知っている。

その感覚を記録し、パターンを見つけ、問いを通じて言葉に近づける。

## なぜこの設計か

身体知は、言語化・数値化される「前」に存在する。
だからベクトル検索（意味の類似度を数値で探す）ではなく、思想の筋を構造として継承する方法を選んだ。

般若心経・五蘊 → 哲学×仏教 → 神経科学3層 → 身体知の言葉

この筋が、Socratic Explorerの応答の中に流れている。
