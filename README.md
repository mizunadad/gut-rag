# 腸で考えるRAG

> 言語化できない身体知を、記録・蒸留・継承するRAGシステム

## コンセプト

「腑に落ちる・気配を読む・なんか違う」——これらは脳機能の異なる部位が処理しているから感じる現象であり、言語化の前に身体が先に知っている。

その感覚を記録し、パターンを見つけ、集合知として継承する。

## ツール

| ツール | 役割 |
|---|---|
| [gut-logger](gut-logger-v2.html) | 「なんか」をその場で残す記録ツール＋パターン可視化 |
| [身体知マップ](gut-brain-map.html) | 10の身体知と脳機能の対応図 |
| [Socratic Explorer](gut-socratic.html) | 絵文字タップで感覚を言語化する問いかけツール |
| [完全構造図](gut-rag-graph-v2.html) | 般若心経×哲学×神経科学×RAGの関係図 |

## 思想の筋

般若心経・五蘊 → 哲学×仏教 → 神経科学3層 → 身体知10語 → RAG実装

## データの取り扱い

- 記録はすべてブラウザのlocalStorageに保存されます（端末外に出ません）
- 「もっと深く探る」機能はAnthropicのAPIを使用します
- APIキーは使用者ご自身でご用意・ご管理ください
- APIに送信されるのは感覚・文脈・パターンの情報のみです
